# Spotify & Music Streaming Data Analysis

This project focuses on cleaning and analyzing music streaming data using Python (Pandas) to explore user engagement and audio features.

## Dataset
The dataset contains information about:
- Artist, Track, Album, Channel
- Views, Likes, Comments, Stream
- Audio features such as Energy, Danceability, Tempo, etc.
- Platform usage (YouTube, Spotify)

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Workflow
1. Load dataset
2. Explore data structure
3. Handle missing values
4. Check duplicate data
5. Standardize string formatting
6. Analyze popularity metrics (Views, Likes, Stream)
7. Analyze platform usage
8. Data visualization

## Key Analysis
- Sorted most popular tracks based on views, likes, and streams
- Identified most used music streaming platforms
- Visualized platform distribution

## How to Run
1. Upload dataset and notebook to Google Colab or Jupyter Notebook
2. Run all cells sequentially
